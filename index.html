🎉 Deploy for free — no credit card, no VPS cost. Takes ~10 minutes.
OPTION A — Railway.appFREE
1
Push your bot to GitHub
Put these files in a GitHub repo: api.py, ai_router.py, trading.py, auth.py, requirements.txt
pip freeze > requirements.txt git init && git add . && git commit -m "initial commit" gh repo create trading-bot --public --push
2
Deploy on Railway.app
Go to railway.app → New Project → Deploy from GitHub → pick your repo. It auto-detects Python.
3
Add your environment variables
In Railway → Variables tab, add these (only the ones you have keys for):
ANTHROPIC_API_KEY=sk-ant-... OPENAI_API_KEY=sk-... GROK_API_KEY=... GEMINI_API_KEY=... GROQ_API_KEY=gsk_... OPENROUTER_API_KEY=sk-or-... HELIUS_API_KEY=... WALLET_PRIVATE_KEY_B58=...
4
Set the start command
In Railway → Settings → Start Command:
uvicorn api:app --host 0.0.0.0 --port $PORT
5
Get your public URL
Railway gives you a URL like https://yourapp.up.railway.app. Paste that in the endpoint field on overview tab and hit connect.
OPTION B — Render.comFREE
1
Same GitHub repo setup as above, then:
Go to render.com → New → Web Service → connect GitHub repo
2
Build & Start commands
Build: pip install -r requirements.txt Start: uvicorn api:app --host 0.0.0.0 --port $PORT
Add env vars in the Environment tab. Free tier sleeps after 15min inactivity — use Railway if you want it always-on.
OPTION C — Local only (no deploy)
1
Run on your laptop, share via ngrok
cd ~/trading-bot source venv/bin/activate uvicorn api:app --host 0.0.0.0 --port 8000 # in another terminal: ngrok http 8000
ngrok gives you a public URL like https://abc123.ngrok.io — free, no signup needed. Share that URL with people.

Netlify drag-and-drop — 30 seconds
Go to netlify.com → Add new site → Deploy manually → drag this HTML file into the box. Done. Your users get a URL like https://yourapp.netlify.app. They enter your Railway/Render URL as the endpoint.
